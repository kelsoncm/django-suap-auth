Fluxo OAuth2
  1. Clique em Entrar com o SUAP — você é redirecionado para /login/.
  2. Na página de login, clique no botão Entrar com o SUAP.
  3. O pacote gera um state, salva na sessão e redireciona para o servidor SUAP.
  4. Após autenticar no SUAP, o servidor retorna para /auth/suap/callback/.
  5. O pacote valida o state, troca o código pelo token, busca o perfil e cria/atualiza o usuário Django.
  6. Você é redirecionado para /dashboard/ já autenticado.
